Learning Deep Learning

2025-01-27

Introduction#

To start from I am using this book hosted at https://d2l.ai/chapter_preface/index.html

Machine learning was not known that much back then as much it is known now.

Testing the potential of deep learning presents unique challenges because any single application brings together various disciplines. Applying deep learning requires simultaneously understanding. i) the motivation for casting a problem in a particular way. ii) The mathematical form of a given model iii) The optimization algorithms for fitting the models to a data iv) The statistical principles that tells us when we should expect our models to generalize unseen data and practical methods for certifying that they have in fact generalized. v) the engineering techniques required to train models efficiently, navigating the pitfalls of numerical computing and getting the most of the available hardware.

Installation#

As we’ll be using python as the core language for creating our notebook, we need to have python installed in our machine and also the jupyter notebook.

Also we need to install miniconda - Miniconda, a lightweight distribution of the popular Anaconda Python distribution, provides a powerful solution for managing package dependencies and creating isolated virtual environments.

Download from here - https://docs.anaconda.com/miniconda/install/

Install the pytorch framework -

pip install torch==2.0.0 torchvision==0.15.1

Installing the d2l package that is created by the writers of this book which encapsulates all the commonly used functions and classes found throughout the book.

pip install d2l==1.0.3
